## Requisitos
Tener instalado Node.js en tu sistema operativo.

1. Clonar este repositorio o descargar el zip desde github.
2. Extraer el repositorio

## Instalar dependencias
```npm install``` 
## Ejecución
En la carpeta raíz, ejecutar el siguiente comando:
```npm run dev```
